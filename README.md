For start the game, open moonpatrol.py
For set your language, open setlanguage.py

Instructions:
Player 1:
SU = Directional arrow up
DOWN = Directional arrow down
LEFT = Left directional arrow
RIGHT = Right directional arrow
SPARO = Spacebar
Player 2:
SU = W
DOWN = S
LEFT = A
RIGHT = F
SPARO = Z
